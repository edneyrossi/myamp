# myamp
Tube amplifier bluetooth smarthphone control

Amplificador Valvulado Contrololado por smartphone

Este projeto é direcionado à aqueles que querem  montar um amplificador valvulado (do inglês, Tube Amplifier), e com isso aprender eletrônica, história das válvulas, funcionamento das válvulas, porque o amplificador valvulado tem som melhor que o transistorizado, não se trata de um livro de receitas. Entender como embarcar uma placa Arduino e como conecta-la a um Smartphone para controlar o amplificador valvulado, com tecnologia sem fio. Amplificadores valvulados são a elite dos amplificadores, preferidos por músicos e audiófilos, desde a invenção da eletrônica até os atuais Smartphones. O MyAmp, Amplificador valvulado controlado por smartphone, pretende ser um projeto que sintetiza a história da eletrônica, em uma linha de tempo de 137 anos, que começa quando Thomas Edison, ligou a primeira válvula eletrônica em 1879, e vai até os dias de hoje. Trazendo valores históricos, educacionais e emocionais. É possível controlar um antigo amplificador valvulado por Smartphone? E porque o som do amplificador valvulado parece tão agradável aos nossos ouvidos? Buscou-se a montagem econômica, esse projeto foi possível graças a plataforma de código fonte aberto de Hardware e Software, de computação embarcada utilizada, o Arduino, com expansão de suas funcionalidades através do bluetooth.
